# Automation

Automation templates for businesses.

## Directory Structure

This repository contains workflow automation templates organized by business function:

- **[Sales](./sales/)** - Sales pipeline, quote generation, and CRM integration workflows
- **[Customer Support](./customer-support/)** - Ticket routing, escalation, and support automation
- **[Lead Management](./lead-management/)** - Lead capture, scoring, and qualification workflows
- **[IT Ops](./it-ops/)** - Infrastructure provisioning, monitoring, and incident response automation
- **[Lead Nurturing](./lead-nurturing/)** - Email drip campaigns and engagement tracking workflows

## Getting Started

Navigate to the relevant directory for your automation needs and follow the instructions in each README.
